# Plane-Game-Diego
Duck Hunter 3D es un juego 3D donde el jugador controla una nave que vuela a través de un escenario y debe recoger patitos de goma mientras esquiva bombas. El objetivo es conseguir la mayor puntuación posible antes de chocar con una bomba.  El juego combina reflejos, precisión y atención, dentro de un entorno 3D.
